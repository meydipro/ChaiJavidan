# چای جاویدان | وبسایت فروش چای ممتاز

وبسایت کامل فروش چای با تم طلایی لوکس، انیمیشن‌های نرم iOS-like و پنل ادمین کامل.

## ویژگی‌ها

- **طراحی لوکس طلایی**: لوگوی طلایی فنجان چای با برگ داخل آن
- **انیمیشن‌های نرم**: استفاده از Framer Motion و انیمیشن‌های سبک
- **مگامنو داینامیک**: دسته‌بندی‌ها و فیلتر بر اساس استان (گیلان و بوشهر)
- **صفحات کامل**: خانه، فروشگاه، جزئیات محصول، درباره، تماس، سبد خرید
- **فیلترها**: بر اساس استان و نوع چای (سیاه، سبز، سفید، طعم‌دار)
- **پنل ادمین قدرتمند**:
  - لاگین امن (ایمیل + رمز)
  - ورود با گوگل (دمو)
  - مدیریت کامل محصولات (افزودن، ویرایش، حذف)
  - مدیریت محتوای سایت (هیرو، درباره ما و ...)
  - مشاهده سفارش‌ها
- **React + Tailwind + Node.js**
- **سبک و قابل اجرا روی دستگاه ضعیف**: انیمیشن‌های سبک و scroll reveal

## راه‌اندازی

### ۱. نصب و اجرا

```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend (در ترمینال جدا)
cd backend
npm install
npm run dev
```

### ۲. تنظیمات

- Backend روی `http://localhost:5001` اجرا می‌شود
- Frontend روی `http://localhost:5173`
- برای دسترسی به پنل ادمین:
  - به `/admin` بروید
  - لاگین با: `admin@chaijavidan.com` / `admin123`

### ۳. لوگو

لوگوی طلایی آماده در `/public/logo.png` موجود است.

## ساختار

```
frontend/
  src/
    components/    # Navbar, MegaMenu, ProductCard, ...
    pages/         # Home, Shop, Admin, ...
    context/       # Cart & Admin
backend/
  server.js      # API کامل Node + Express
```

## نکات

- داده‌ها در حال حاضر در حافظه (in-memory) هستند. برای محصول نهایی MongoDB اضافه کنید.
- ورود با گوگل دمو است. می‌توانید با google-auth-library واقعی کنید.
- تمام تغییرات پنل ادمین به صورت واقعی روی API اعمال می‌شود و در تمام صفحات نمایش داده می‌شود.
